# Exam7Solutions
## Setup Project

- Node.js 18, 20 or 22. 
- Require: npm, chrome
- step 1 

```bash
$ npm init playwright@latest
$ yarn create playwright

```
- step 2

```bash
$ npx playwright install

```

## Execute test

Execute command `npm test:<project name>`

Example run test lotteryplus:

```bash
$ npm run "test:lotteryplus"
$ yarn run "test:lotteryplus"

```
