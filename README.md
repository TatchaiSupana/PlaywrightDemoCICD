
## Setup Project

- Node.js 18, 20 or 22. 
- Require: npm, chrome
- step 1 

```bash
- windows OS
$ npm init playwright@latest
- mac OS
$ yarn create playwright

```
- step 2

```bash
$ npx playwright install

```

## Execute test

Execute command `npm test:<project name>`

Example run test empeo:

```bash
- windows OS
$ npm run "test:empeo"
- mac OS
$ yarn run "test:empeo"

```
